# 扑克大战 Poker Battle · 完整规则集（无扩 1.8 版本）

> 基于原版规则书（BY -Random 2024.9.27）整理，保留完整规则细节。

---

## 一、胜利目标

击毁对方**大本营**（即耗尽对方牌库）时，游戏结束，攻击方胜利。

---

## 二、双人起始配置

**道具**：一副去除大小王的标准 52 张扑克牌。

### 发牌

1. 每名玩家获得 **7 张手牌** + **13 张牌库**（牌库对双方均不可见内容）
2. 剩余 **12 张**形成**墓地**（公共区域）

### 开局流程

1. **换牌**：从先手方开始，选择至多 **3 张**手牌与墓地中的牌对换
2. **初始部署**：从先手方开始，各自执行一个部署阶段（手牌 → 待战区）
3. **正式回合开始**：先手方的**第一个行动回合仅有 1 点行动力**，随后正常游戏

---

## 三、配置区域

每位玩家拥有以下区域：

```
        [作战区 0] [作战区 1] [作战区 2]    ← 前排（直接参战）
        [待战区 0] [待战区 1] [待战区 2]    ← 后排（部署目的地）
        [手牌]                              ← 仅自己可见
        [牌库/大本营]                        ← 对双方均不可见
```

公共区域：
```
        [墓地]                              ← 所有弃置的牌汇入此处
```

**部队**：任意一个作战区或待战区中的所有牌称为 1 个部队。每个部队的卡牌数量**至多为 5 张**。

---

## 四、完整回合流程

每个回合包含三个阶段：

### 1. 准备阶段

将待战区的部队**不变顺序地**前移至对应作战区：
- 可以选择全部前移
- 也可以选择将待战区中**上方任意张**卡牌不变顺序地移至对应作战区现有部队**下方**

### 2. 行动阶段

每回合 **2 点行动力**（先手首回合仅 1 点），每个行动消耗 1 点。**每种行动每回合限一次**，未消耗的行动力不能储存。

| # | 行动 | 效果 |
|---|---|---|
| 1 | **征兵** | 从墓地抽 **2 张**牌到己方牌库 |
| 2 | **训练** | 从己方牌库抽 **2 张**牌至己方手牌 |
| 3 | **重编** | 合并**相邻**作战区的部队，调整牌序，以任意顺序、任意数量分配到原先两格中（单个部队不超过 5 张） |
| 4 | **制衡** | 模式 A：选择 1 张手牌与己方牌库中 1 张牌交换，至多重复 **4 次**；模式 B：将至多 **2 张**手牌与墓地中 2 张牌交换，至多重复 **1 次** |
| 5 | **进攻** | 使用一支部队**顶端**的 1-2 张牌发起攻击（详见第五章） |

### 3. 部署阶段

从手牌中以**任意顺序**，放置**任意张**卡牌至待战区。如果待战区仍有部队，则只能置于其**下方**。

---

## 五、攻击流程

### 第 1 步：确定攻击牌

进攻方从一支部队**顶端**翻开 **1-2 张**牌作为进攻牌。

### 第 2 步：确定攻击目标 & 计算攻击点数

- 如果对应列敌军**作战区有部队** → 必须先攻击作战区部队
- 否则进攻方可选择：攻击**待战区部队** 或 直接进攻**大本营**

**点数规则**：
- 数字牌 = 对应点数（2-10）
- J/Q/K = **10** 点
- A = 玩家选择 **1 点**或 **11 点**

### 第 3 步：防守

防守方从上至下**依次亮开牌**进行防守。所有亮出牌的点数之和 **≥** 进攻方点数 → 防守成功。

> 即使防守点数已大于进攻点数，防守方仍然**可以选择继续翻开**同部队的剩余牌（目的：触发更多弃牌效果）。

### 第 4 步：击毁 & 溢出

若一支部队亮出**所有牌**后点数之和仍**小于**进攻点数：
- 该部队被**击毁**
- 被击毁的部队**跳过所有弃牌效果**
- 攻方溢出的点数可对敌方**待战区部队**或**大本营**执行**溢出攻击**
- 溢出攻击待战区时，直接将待战区的牌算作新部队的牌，但其黑桃**不执行沉默效果**

### 第 5 步：弃牌效果

确定本次战斗双方阵亡（参与攻防）的牌，从**攻方开始**执行弃牌效果。

### 第 6 步：清理

弃置所有参与本次攻防的牌进入**墓地**，洗混墓地。

## 六、花色效果与结算规则

### 基础规则

#### ♣ 梅花（草花）—— 剑

- 仅在**进攻结算**时有效果
- 效果：使本次攻击中**所有梅花牌**的点数之和 **×2**
- 受"第一张牌花色为准"规则约束（见下方第 3 条）

#### ♠ 黑桃 —— 盾

- 仅在**防守结算**时有效果
- 效果：无论黑桃牌在何时加入防御，都**可以选择**将该黑桃牌计入防守点数时 **×2**
- **沉默**：若（1）首张亮出的防御牌为黑桃，且（2）进攻方总点数 < 所有亮出的黑桃牌点数之和 → **无效化**攻击方本轮进攻的弃牌效果

#### ♥ 红桃 —— 恢复

- 在**被弃置时**产生效果
- **弃牌效果**：从己方牌库中抽取至多 **X 张**牌加入手牌
- **急救**（特殊）：作为**防守方被翻出时**，防守方可立即从手牌中打出**任意 1 张牌**加入本次防守。每轮防守**限一次**。若执行急救效果，则触发急救效果的这张红桃牌**无法触发弃牌效果**

#### ♦ 方片 —— 整备

- 在**被弃置时**产生效果
- **弃牌效果**：从墓地抽取 **X 张**牌加入己方牌库，并可**调整当前部队排序**

#### X 值计算

| 牌点数 | X 值 |
|---|---|
| 2、3、4、5 | 3 |
| 6、7、8、9 | 2 |
| 10、J、Q、K | 1 |
| A（作为 1 点时） | 4 |
| A（作为 11 点时） | 1 |

> 弃牌多张时 X **累加**计算，但 X **至多为 6**。

### "第一张牌花色为准"规则

- **攻击方**：执行的花色效果均以战斗时翻开的**第一张牌花色**为准
  - 例：第一张是红桃，第二张是梅花 → 梅花**不执行翻倍**
  - 例：两张都是梅花 → 第二张梅花也执行翻倍
- **防守方**：执行的**弃牌效果**以翻开的**第一张牌花色**为准
  - 例：第一张是黑桃 → 后续红桃/方片**不结算弃牌效果**

### 大本营防守惩罚

进攻大本营时，直接从防守方**牌库**中抽取牌直至点数和 ≥ 进攻点数：

1. **不执行任何花色效果**
2. **A 视作 1 点**
3. **所有原先为 10 点的牌（10/J/Q/K）视为 5 点**

> 大本营每张牌的点数期望 = 260÷52 = **5**

若牌库抽完点数和仍小于进攻点数，防守方**告负**（游戏结束）。

---

## 七、特殊规则（可约定是否加入）

### 特殊规则 1：四象防御

若一轮防守中亮出过**四张花色不同**的防守牌：
- 在最后一门花色亮出时，**立即视为防守成功**（无论点数）
- 结算时**同时执行所有**参与防守的方片和红桃牌的弃牌效果，无论其是否作为第一张牌被亮出
- 但如果参与防守的红桃牌已经执行过急救效果，**仍然不执行其弃牌效果**

### 特殊规则 2：连张

无论是进攻还是防守，连续亮出**任意张点数一样**的牌称作连张：
- 使用者可**自定义这些连张的花色**（需要是连张中有的花色）
- 例：防守方亮出 8♠ 和 8♥ → 可声明两张牌花色均为红桃，或一张红桃一张黑桃，并执行对应花色效果

## 附录：关键名词

| 名词 | 定义 |
|---|---|
| **部队** | 作战区或待战区中的一叠牌（1-5张） |
| **击毁** | 一支部队所有牌亮出后点数之和仍 < 攻击点数 |
| **沉默** | 首张防御牌为黑桃，且黑桃总和 ≥ 攻击 → 无效化攻击方弃牌效果 |
| **急救** | 防御翻出红桃时，可从手牌打出 1 张加入防御（每轮限一次） |
| **溢出** | 击毁后剩余攻击力，可贯穿到下一层（待战区 / 大本营） |
| **大本营** | 牌库的别称，被攻击时受惩罚规则约束 |
| **墓地** | 所有弃置的牌汇入此处（战斗后洗混） |

## 附录：回合流程速查

```
开局：发牌(7+13+12) → 换牌(≤3) → 初始部署 → 正式开始

每回合：
  ① 准备阶段：待战区 → 作战区（可选部分/全部前移）
  ② 行动阶段：2点行动力（先手首回合1点），5种操作各限一次
     - 征兵 / 训练 / 重编 / 制衡 / 进攻
  ③ 部署阶段：手牌 → 待战区底部
```

```
攻击流程：
  ① 攻击方翻1-2张（顶端）
  ② 确定目标 + A点数 → 计算总攻击（♣翻倍受"第一张花色"约束）
  ③ 防守方依次翻牌（♠可选×2 / ♥可急救）直到点数≥攻击 或 部队耗尽
  ④ 耗尽=击毁 → 跳过弃牌效果 + 溢出攻击
  ⑤ 弃牌效果结算（攻方先 → 防方）
  ⑥ 全部参与牌入墓地，洗混
```
